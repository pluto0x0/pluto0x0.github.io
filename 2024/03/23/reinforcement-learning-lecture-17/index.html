A Question

Es,r,s′[(Vθ(s)−r−γVθ(s′))2]\mathbb{E}_{s,r,s^{\prime}}\left[\left(V_\theta(s)-r-\gamma V_\theta\left(s^{\prime}\right)\right)^2\right]Es,r,s′​[(Vθ​(s)−r−γVθ​(s′))2]

We do Vθ(s)←Vθ(s)+α(r−γVθ(s′)−Vθ(s))V_\theta(s)\leftarrow V_\theta(s) + \alpha(r-\gamma V_\theta\left(s^{\prime}\right) - V_\theta(s))Vθ​(s)←Vθ​(s)+α(r−γVθ​(s′)−Vθ​(s)) in TD(0).

What if we minimize the square error between Vθ(s)V_\theta(s)Vθ​(s) and its target, i.e. Es,r,s′[(Vθ(s)−r−γVθ(s′))2]\mathbb{E}_{s,r,s^{\prime}}\left[\left(V_\theta(s)-r-\gamma V_\theta\left(s^{\prime}\right)\right)^2\right]Es,r,s′​[(Vθ​(s)−r−γVθ​(s′))2] ?

No correct. It can be decomposed as the sum of 2 parts:

  • Es[(Vθ(s)−(TπVθ)(s))2]\mathbb{E}_s\left[\left(V_\theta(s)-\left(\mathcal{T}^\pi V_\theta\right)(s)\right)^2\right]Es​[(Vθ​(s)−(TπVθ​)(s))2]
    • good. It’s L-2 norm Bellman Error.
  • γ2Es[Var⁡s′∣s,π(s)[Vθ(s′)]]\gamma^2 \mathbb{E}_s\left[\operatorname{Var}_{s^{\prime} \mid s, \pi(s)} \left[ V_\theta\left(s^{\prime}\right)\right]\right]γ2Es​[Vars′∣s,π(s)​[Vθ​(s′)]]
    • Not good. It penalize policy with large variance.
    • OK for deterministic environment because the variance is always 000 in this case.

Solution

If we have a simulator, for each sss in data, draw another independent state transition.

Minimize objective

E[(Vθ(s)−r−γVθ(sA′))(Vθ(s)−r−γVθ(sB′)]\mathbb{E}\left[\left(V_\theta(s)-r-\gamma V_\theta\left(s_A^{\prime}\right)\right)\left(V_\theta(s)-r-\gamma V_\theta\left(s_B^{\prime}\right)\right]\right. E[(Vθ​(s)−r−γVθ​(sA′​))(Vθ​(s)−r−γVθ​(sB′​)]

“Double sampling” and Baird’s residual algorithm (Bellman residual minimization).

Convergence

  • TD with function approximation can diverge in general
  • Is it because of…
    • Randomness in SGD?
      • Nope. Even the batch version doesn’t converge
    • Sophisticated, non-linear func approx?
      • Nope. Even linear doesn’t converge.
    • That our function class does not capture V"?
      • Nope. Even if V" can be exactly represented in the function class (“realizable”), it still does not converge.
