Reinforcement Learning (2-4)

Markov Decision Processes

Infinite-horizon discounted MDPs

An MDP M=(S,A,P,R,γ)M = (S, A, P, R, \gamma)M=(S,A,P,R,γ) consists of:

  • State space SSS .
  • Action space AAA .
  • Transition function PPP : S×A→Δ(S)S \times A \rightarrow \Delta(S)S×A→Δ(S) . Δ(S)\Delta(S)Δ(S) is the probability simplex over SSS , i.e., all non-negative vectors of length ∣S∣| S|∣S∣ that sums up to 111 .
  • Reward function RRR : S×A→RS \times A \rightarrow \mathbb{R}S×A→R . (deterministic reward function)
  • Discount factor γ∈[0,1]\gamma \in [0,1]γ∈[0,1]

The agent

  1. starts in some state s1s_1s1​
  2. takes action a1a_1a1​
  3. receives reward r1=R(s1,a1)r_1 = R(s_1, a_1)r1​=R(s1​,a1​)
  4. transitions to s2∼P(s1,a1)s_2 \sim P(s_1, a_1)s2​∼P(s1​,a1​)
  5. takes action a2a_2a2​
  6. … and so on so forth — the process continues forever.

Objective: (discounted) expected total reward

  • Other terms used: return, value, utility, long-term reward, etc

Example: Gridworld

  • State: grid x, y
  • Action: N, S, E, W
  • Dynamics:
    • most cases: move to adjacent grid
    • meet wall or reached goal: keep in the current state
  • Reward:
    • 000 in the goal state
    • −1-1−1 everywhere else
  • Discount factor γ\gammaγ : 0.99

For γ<1\gamma < 1γ<1 , the total reward is finite.

finite horizon vs. infinite-horizon discounted MDP

  • For finite-horizon (finite acitons), γ\gammaγ can be 111 .
  • For infinite-horizon (infinite acitons), γ<1\gamma < 1γ<1 .
