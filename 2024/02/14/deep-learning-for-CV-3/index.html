Support vector machine (SVM)

maximize the distance between the hyperplane
and the closest training example, where the distance is given by ∣wTx0∣∥w∥\frac{\left|w^T x_0\right|}{\|w\|}∥w∥∣wTx0​∣​ .

finding hyperplane

Assuming the data is linearly separable, we can fix the scale of www so that yiwTxi=1y_i w^T x_i=1yi​wTxi​=1 for support vectors and yiwTxi≥1y_i w^T x_i \geq 1yi​wTxi​≥1 for all other points.

i.e. We want to maximize margin 1∥w∥\frac{1}{\|w\|}∥w∥1​ while correctly classifying all training data: yiwTxi≥1y_i w^T x_i \geq 1yi​wTxi​≥1 , or

min⁡w12∥w∥2 s.t. yiwTxi≥1∀i.\min _w \frac{1}{2}\|w\|^2 \quad \text { s.t. } \quad y_i w^T x_i \geq 1 \quad \forall i. wmin​21​∥w∥2 s.t. yi​wTxi​≥1∀i.

General recipe

empirical loss = empirical loss + data loss

L^(w)=λR(w)+1n∑i=1nl(w,xi,yi)\hat{L}(w) \quad=\lambda R(w)+\frac{1}{n} \sum_{i=1}^n l\left(w, x_i, y_i\right) L^(w)=λR(w)+n1​i=1∑n​l(w,xi​,yi​)

regularization

  • L2 regularization: R(w)=12∥w∥22R(w)=\frac{1}{2}\|w\|_2^2R(w)=21​∥w∥22​
  • L1 regularization: R(w)=12∥w∥1:=∑d∣w(d)∣R(w)=\frac{1}{2}\|w\|_1 := \sum_d\left|w^{(d)}\right|R(w)=21​∥w∥1​:=∑d​​w(d)​

The gradient of loss function with L1 regularization is

∇L^(w)=λsgn⁡(w)+∑i=1n∇l(w,xi,yi)\nabla \hat{L}(w)=\lambda \operatorname{sgn}(w)+\sum_{i=1}^n \nabla l\left(w, x_i, y_i\right) ∇L^(w)=λsgn(w)+i=1∑n​∇l(w,xi​,yi​)

L1 regularization encourages sparsity weight.
